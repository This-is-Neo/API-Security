    API3:2019 Ш§ЩҒШҙШ§ЫҢ Щ…ЩҒШұШ· ШҜШ§ШҜЩҮ

    Ш№ЩҲШ§Щ…Щ„ ШӘЩҮШҜЫҢШҜ/Щ…ШіЫҢШұ ШӯЩ…Щ„ЩҮ Ш¶Ш№ЩҒ Ш§Щ…ЩҶЫҢШӘЫҢ ЩҫЫҢШ§Щ…ШҜ
    API Ш®Ш§Шө: ЩӮШ§ШЁЩ„ЫҢШӘ ШЁЩҮШұЩҮвҖҢШЁШұШҜШ§ШұЫҢ3 Щ…ЫҢШІШ§ЩҶ ШҙЫҢЩҲШ№2 : ЩӮШ§ШЁЩ„ЫҢШӘ ШӘШҙШ®ЫҢШө2 ЩҫЫҢШ§Щ…ШҜ ЩҒЩҶЫҢ2 : Ш®Ш§Шө Ъ©ШіШЁ ЩҲ Ъ©Ш§Шұ
    ШЁЩҮШұЩҮ ШЁШұШҜШ§ШұЫҢ Ш§ШІ Ш§ЫҢЩҶ ШўШіЫҢШЁвҖҢЩҫШ°ЫҢШұЫҢ ШўШіШ§ЩҶ ШЁЩҲШҜЩҮ ЩҲ ШәШ§Щ„ШЁШ§ ШЁШ§ ШҙЩҶЩҲШҜ ШӘШұШ§ЩҒЫҢЪ© ШЁЩҮ Щ…ЩҶШёЩҲШұ ШӘШӯЩ„ЫҢЩ„ ЩҫШ§ШіШ®вҖҢЩҮШ§ЫҢ API ШЁШұШ§ЫҢ ЫҢШ§ЩҒШӘЩҶ ШҜШ§ШҜЩҮ ШӯШіШ§ШіЫҢ Ъ©ЩҮ ЩҶШЁШ§ЫҢШҜ ШЁЩҮ Ъ©Ш§ШұШЁШұ ШЁШ§ШІЪҜШұШҜШ§ЩҶШҜЩҮ ШҙЩҲШҜ Ш§Щ…Ъ©Ш§ЩҶ ЩҫШ°ЫҢШұ Ш§ШіШӘ. APIЩҮШ§ ШЁШұШ§ЫҢ ЩҒЫҢЩ„ШӘШұ Ъ©ШұШҜЩҶ ШҜШ§ШҜЩҮ ШЁЩҮ Ъ©Щ„Ш§ЫҢЩҶШӘвҖҢЩҮШ§ Ш§ШӘЪ©Ш§ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜ. Ш§ШІ ШўЩҶШ¬Ш§ Ъ©ЩҮ APIЩҮШ§ ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ Щ…ЩҶШ§ШЁШ№ ШҜШ§ШҜЩҮ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲЩҶШҜШҢ ШӘЩҲШіШ№ЩҮ ШҜЩҮЩҶШҜЪҜШ§ЩҶ ЪҜШ§ЩҮШ§ ШўЩҶЩҮШ§ ШұШ§ ШЁШҜЩҲЩҶ ШӘЩҲШ¬ЩҮ ШЁЩҮ ШӯШіШ§ШіЫҢШӘ Ш§Ш·Щ„Ш§Ш№Ш§ШӘЫҢ Ъ©ЩҮ Ш§ЩҒШҙШ§ Щ…ЫҢвҖҢШҙЩҲШҜ ШЁЪ©Ш§Шұ Щ…ЫҢвҖҢЪҜЫҢШұЩҶШҜ. Ш§ШЁШІШ§ШұЩҮШ§ЫҢ Ш®ЩҲШҜЪ©Ш§Шұ ШәШ§Щ„ШЁШ§ ЩҶЩ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ Ш§ЫҢЩҶ вҖҢШўШіЫҢШЁвҖҢЩҫШ°ЫҢШұЫҢ ШұШ§ Ъ©ШҙЩҒ Ъ©ЩҶЩҶШҜ ЪҶШұШ§ Ъ©ЩҮ ШӘЩ…Ш§ЫҢШІ ШҜШ§ШҜЩҶ ШЁЫҢЩҶ ШҜШ§ШҜЩҮ Щ…Ш¬Ш§ШІЫҢ Ъ©ЩҮ ШӘЩҲШіШ· API ШЁШ§ШІЪҜШұШҜШ§ЩҶШҜЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ ШЁШ§ ШҜШ§ШҜЩҮ ШӯШіШ§ШіЫҢ Ъ©ЩҮ ЩҶШЁШ§ЫҢШҜ ШӘЩҲШіШ· API ШЁШ§ШІЪҜШұШҜШ§ЩҶШҜЩҮ ШҙЩҲШҜ ШЁШҜЩҲЩҶ ШҜШ§ШҙШӘЩҶ ЩҒЩҮЩ…ЫҢ Ш№Щ…ЫҢЩӮ Ш§ШІ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶ Ш§Щ…Ъ©Ш§ЩҶ ЩҫШ°ЫҢШұ ЩҶЫҢШіШӘ. Ш§ЩҒШҙШ§ЫҢ Щ…ЩҒШұШ· ЩҲ ШЁЫҢШҙ Ш§ШІ ШӯШҜ ШҜШ§ШҜЩҮ Щ…Ш№Щ…ЩҲЩ„Ш§ Щ…ЩҶШ¬Шұ ШЁЩҮ Ш§ЩҒШҙШ§ЫҢ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ ШӯШіШ§Ші Щ…ЫҢвҖҢШҙЩҲШҜ.

    ШўЫҢШ§ API Ш§ШІ ЩҶШёШұ Ш§ЩҒШҙШ§ЫҢ Щ…ЩҒШұШ· ШҜШ§ШҜЩҮ вҖҢШўШіЫҢШЁвҖҢЩҫШ°ЫҢШұ Ш§ШіШӘШҹ

    Ш·ШұШ§ШӯЫҢ API ШЁЩҮ ЪҜЩҲЩҶЩҮвҖҢШ§ЫҢ Ш§ШіШӘ Ъ©ЩҮ ШҜШ§ШҜЩҮ ШӯШіШ§Ші ШұШ§ ШЁЩҮ Ъ©Щ„Ш§ЫҢЩҶШӘ ШЁШ§ШІ Щ…ЫҢвҖҢЪҜШұШҜШ§ЩҶШҜ. Ш§ЫҢЩҶ ШҜШ§ШҜЩҮ ШәШ§Щ„ШЁШ§ ЩҫЫҢШҙ Ш§ШІ Ш§ШұШ§ШҰЩҮ ЩҲ ЩҶЩ…Ш§ЫҢШҙ ШЁЩҮ Ъ©Ш§ШұШЁШұ ШҜШұ ШіЩ…ШӘ Ъ©Щ„Ш§ЫҢЩҶШӘ ЩҒЫҢЩ„ШӘШұ Щ…ЫҢвҖҢШҙЩҲШҜ. ШҜШұ ЩҶШӘЫҢШ¬ЩҮ Щ…ЩҮШ§Ш¬Щ… Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШЁШұШ§ШӯШӘЫҢ ЩҲ ШЁШ§ ШҙЩҶЩҲШҜ ШӘШұШ§ЩҒЫҢЪ©ШҢ Ш§ЫҢЩҶ ШҜШ§ШҜЩҮ ШӯШіШ§Ші ШұШ§ Щ…ШҙШ§ЩҮШҜЩҮ ЩҶЩ…Ш§ЫҢШҜ.

    Щ…Ш«Ш§Щ„вҖҢЩҮШ§ЫҢЫҢ Ш§ШІ ШіЩҶШ§ШұЫҢЩҲЩҮШ§ЫҢ ШӯЩ…Щ„ЩҮ

    ШіЩҶШ§ШұЫҢЩҲ #1

    ШӘЫҢЩ… ШӘЩҲШіШ№ЩҮ Щ…ЩҲШЁШ§ЫҢЩ„ Ш§ШІ /api/articles/{articleId}/comments/{commentId} ШЁШұШ§ЫҢ Щ…ШҙШ§ЩҮШҜЩҮ ЩҲ ЩҫШұШҜШ§ШІШҙ ЩҒШұШ§ШҜШ§ШҜЩҮ Ъ©Ш§Щ…ЩҶШӘвҖҢЩҮШ§ ШЁЩҮШұЩҮ Щ…ЫҢвҖҢШЁШұШҜ. ШЁШ§ ШҙЩҶЩҲШҜ ШӘШұШ§ЩҒЫҢЪ© Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶ Щ…ЩҲШЁШ§ЫҢЩ„ШҢ Щ…ЩҮШ§Ш¬Щ… ШҜШұ Щ…ЫҢвҖҢЫҢШ§ШЁШҜ Ъ©ЩҮ ШҜШ§ШҜЩҮ Щ…ШұШӘШЁШ· ШЁШ§ ЩҶЩҲЫҢШіЩҶШҜЩҮ Ъ©Ш§Щ…ЩҶШӘ ЩҶЫҢШІ ШЁШ§ШІЪҜШұШҜШ§ЩҶШҜЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ. Ш§ЫҢЩҶ Щ…ЩҲШ¶ЩҲШ№ ШЁЩҮ Ш§ЫҢЩҶ ШҜЩ„ЫҢЩ„ Ш§ШіШӘ Ъ©ЩҮ вҖҢвҖҢЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ API Ш§ШІ ЫҢЪ© Щ…ШӘШҜ Ш№Щ…ЩҲЩ…ЫҢ toJSON() ШЁШұШ§ЫҢ ШіШұЫҢШ§Щ„ЫҢШІЩҮ Ъ©ШұШҜЩҶ ШҙЫҢШЎ User ШЁЩҮШұЩҮ Щ…ЫҢвҖҢШЁШұШҜ Ъ©ЩҮ Ш§ЫҢЩҶ ШҙЫҢ ШӯШ§ЩҲЫҢ ШҜШ§ШҜЩҮ ШӯШіШ§Ші PII Щ…ЫҢ ШЁШ§ШҙШҜ.

    ШіЩҶШ§ШұЫҢЩҲ #2

    ЫҢЪ© ШіЫҢШіШӘЩ… ЩҶШёШ§ШұШӘЫҢ Щ…ШЁШӘЩҶЫҢ ШЁШұ IoT ШЁЩҮ Щ…ШҜЫҢШұШ§ЩҶ Ш®ЩҲШҜ Ш§Ш¬Ш§ШІЩҮ Ш§ЫҢШ¬Ш§ШҜ Ъ©Ш§ШұШЁШұШ§ЩҶЫҢ ШЁШ§ ШіШ·ЩҲШӯ Щ…Ш¬ЩҲШІ Щ…Ш®ШӘЩ„ЩҒ Щ…ЫҢвҖҢЩҶЩ…Ш§ЫҢШҜ. ЫҢЪ©ЫҢ Ш§ШІ Щ…ШҜЫҢШұШ§ЩҶ ЫҢЪ© ШӯШіШ§ШЁ Ъ©Ш§ШұШЁШұЫҢ ШЁШұШ§ЫҢ ЫҢЪ© ЩҶЫҢШұЩҲЫҢ ШӯЩҒШ§ШёШӘ ЩҒЫҢШІЫҢЪ©ЫҢ (ЩҶЪҜЩҮШЁШ§ЩҶ) Ш¬ШҜЫҢШҜ Щ…ЫҢвҖҢШіШ§ШІШҜ Ъ©ЩҮ ШЁШұ Щ…ШЁЩҶШ§ЫҢ ШўЩҶ ШӘЩҶЩҮШ§ Ш§Щ…Ъ©Ш§ЩҶ ШҜШіШӘШұШіЫҢ ШЁЩҮ ШіШ§Ш®ШӘЩ…Ш§ЩҶвҖҢЩҮШ§ЫҢ Щ…ШҙШ®ШөЫҢ ШЁШ§ЫҢШіШӘЫҢ ЩҲШ¬ЩҲШҜ ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙШҜ. ШЁЩҮ Щ…ШӯШ¶ Ш§ШіШӘЩҒШ§ШҜЩҮ ЩҶЪҜЩҮШЁШ§ЩҶ Щ…Ш°ШЁЩҲШұ Ш§ШІ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶ Щ…ЩҲШЁШ§ЫҢЩ„ Ш®ЩҲШҜШҢ ЫҢЪ© ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ API ШЁЩҮ ШіЩҲЫҢ /api/sites/111/cameras ШұЩҲШ§ЩҶЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ ШӘШ§ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ Щ…ШұШӘШЁШ· ШЁШ§ ШҜЩҲШұШЁЫҢЩҶвҖҢЩҮШ§ЫҢ Щ…ЩҲШ¬ЩҲШҜ ШұШ§ ШҜШұЫҢШ§ЩҒШӘ ЩҶЩ…ЩҲШҜЩҮ ЩҲ ШўЩҶЩҮШ§ ШұШ§ ШҜШұ ШҜШҙШЁЩҲШұШҜ Ш®ЩҲШҜ ЩҶЩ…Ш§ЫҢШҙ ШҜЩҮШҜ. ЩҫШ§ШіШ®ШҢ Щ„ЫҢШіШӘЫҢ Ш§ШІ Ш¬ШІШҰЫҢШ§ШӘ ШҜЩҲШұШЁЫҢЩҶвҖҢЩҮШ§ ШЁШ§ ЩҒШұЩ…ШӘ ШІЫҢШұ ШұШ§ ШҜШұ ШЁШұШҜШ§ШұШҜ. {"id":"xxx","live_access_token":"xxxx-bbbbb","building_id":"yyy"}. ШҜШұ ШӯШ§Щ„ЫҢЪ©ЩҮ ШұШ§ШЁШ· ЪҜШұШ§ЩҒЫҢЪ©ЫҢ Ъ©Щ„Ш§ЫҢЩҶШӘ ЩҒЩӮШ· ШҜЩҲШұШЁЫҢЩҶвҖҢЩҮШ§ЫҢЫҢ Ъ©ЩҮ ЩҶЪҜЩҮШЁШ§ЩҶ Щ…Ш°ШЁЩҲШұ ШЁШ§ЫҢШіШӘЫҢ ШЁЩҮ ШўЩҶЩҮШ§ ШҜШіШӘШұШіЫҢ ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙШҜ ШұШ§ ЩҶШҙШ§ЩҶ Щ…ЫҢвҖҢШҜЩҮШҜШҢ Ш§Щ…Ш§ Щ„ЫҢШіШӘ Ъ©Ш§Щ…Щ„ Ш§ЫҢЩҶ ШҜЩҲШұШЁЫҢЩҶвҖҢЩҮШ§ ШҜШұ ЩҫШ§ШіШ® API ЩҲШ¬ЩҲШҜ ШҜШ§ШұШҜ.

    ЪҶЪҜЩҲЩҶЩҮ Ш§ШІ вҖҢШўШіЫҢШЁвҖҢЩҫШ°ЫҢШұЫҢ Ш§ЩҒШҙШ§ЫҢ Щ…ЩҒШұШ· ШҜШ§ШҜЩҮ ЩҫЫҢШҙЪҜЫҢШұЫҢ Ъ©ЩҶЫҢЩ…Шҹ

    • Ш№ШҜЩ… ШӘЪ©ЫҢЩҮ ШЁШұ Ъ©Щ„Ш§ЫҢЩҶШӘ ШҜШұ Щ…ШіШ§Щ„ЩҮ ЩҒЫҢЩ„ШӘШұ Ъ©ШұШҜЩҶ ШҜШ§ШҜЩҮ ШӯШіШ§Ші.
    • ШЁШ§ШІШЁЫҢЩҶЫҢ ЩҫШ§ШіШ® ШҜШұЫҢШ§ЩҒШӘЫҢ Ш§ШІ API ШЁЩҮ Щ…ЩҶШёЩҲШұ ШӯШөЩҲЩ„ Ш§Ш·Щ…ЫҢЩҶШ§ЩҶ Ш§ШІ ШўЩҶЪ©ЩҮ ЩҒЩӮШ· ШҜШ§ШҜЩҮ Щ„Ш§ШІЩ… ЩҲ Ш§ШөЩ„ЫҢ ШҜШұ ШўЩҶ ЩҶЩ…Ш§ЫҢШҙ ШҜШ§ШҜЩҮ Щ…ЫҢ ШҙЩҲШҜ.
    • ЩҫЫҢШҙ Ш§ШІ Ш§ЩҒШҙШ§ ЩҲ ШҜШұ Щ…Ш№ШұШ¶ ШҜЫҢШҜ Ш№Щ…ЩҲЩ… ЩӮШұШ§ШұШҜШ§ШҜЩҶ ЫҢЪ© APIШҢ Щ…ЩҮЩҶШҜШіЫҢЩҶ ШӘЩҲШіШ№ЩҮ ШҜЩҮЩҶШҜЪҜШ§ЩҶ Back-End ШЁШ§ЫҢШіШӘЫҢ Ш§ШІ Ш®ЩҲШҜ ШЁЩҫШұШіЩҶШҜ: Щ…ШөШұЩҒ Ъ©ЩҶЩҶШҜЩҮ ЩҲ Щ…Ш®Ш§Ш·ШЁ Ш§ЫҢЩҶ ШҜШ§ШҜЩҮ ЪҶЩҮ Ъ©ШіЫҢ Ш§ШіШӘШҹ
    • Ш§Ш¬ШӘЩҶШ§ШЁ Ш§ШІ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Щ…ШӘШҜЩҮШ§ЫҢ Ш№Щ…ЩҲЩ…ЫҢ to_json() ЩҲ to_string() ЩҲ ШҜШұ Ш№ЩҲШ¶ ШҜШіШӘЪҶЫҢЩҶ Ъ©ШұШҜЩҶ ШӘЪ© ШӘЪ© ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ ЩҲ Щ…ШҙШ®ШөЩҮвҖҢЩҮШ§ЫҢЫҢ Ъ©ЩҮ ШЁШұШ§ЫҢ ЩҫШ§ШіШ® Ш¶ШұЩҲШұЫҢ ЩҮШіШӘЩҶШҜ.
    • Ш·ШЁЩӮЩҮ ШЁЩҶШҜЫҢ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ ШӯШіШ§Ші ЩҲ ШҙШ®ШөЫҢ Ш°Ш®ЫҢШұЩҮ ШҙШҜЩҮ ШҜШұ APIЩҮШ§ ЩҲ ШЁШ§ШІШЁЫҢЩҶЫҢ ШӘЩ…Ш§Щ…ЫҢ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢвҖҢЩҮШ§ЫҢ APIЩҮШ§ЫҢЫҢ Ъ©ЩҮ Ш§ЫҢЩҶ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ ШұШ§ ШЁШ§ШІ Щ…ЫҢвҖҢЪҜШұШҜШ§ЩҶЩҶШҜ ШЁЩҮ Щ…ЩҶШёЩҲШұ Ъ©ШҙЩҒ ЩҲ ШҙЩҶШ§ШіШ§ЫҢЫҢ Щ…ЩҲШ§ШұШҜЫҢ Ъ©ЩҮ Ш¶Ш№ЩҒ Ш§Щ…ЩҶЫҢШӘЫҢ ШҜШұ ЩҫЫҢ ШҜШ§ШұЩҶШҜ.
    • ШЁЪ©Ш§ШұЪҜЫҢШұЫҢ ЫҢЪ© Щ…Ъ©Ш§ЩҶЫҢШІЩ… Ш§Ш№ШӘШЁШ§ШұШіЩҶШ¬ЫҢ Ш§Щ„ЪҜЩҲЩ…ШӯЩҲШұ ШЁШұШ§ЫҢ ШЁШұШұШіЫҢ Ш§Ш№ШӘШЁШ§Шұ ЩҫШ§ШіШ®вҖҢЩҮШ§ ШЁШ№ЩҶЩҲШ§ЩҶ ЫҢЪ© Щ„Ш§ЫҢЩҮ Ш§Щ…ЩҶЫҢШӘЫҢ ШҜЫҢЪҜШұ ЩҲ ЩҮЩ…ЪҶЩҶЫҢЩҶ ШӘШ№ШұЫҢЩҒ ЩҲ Ш§Ш№Щ…Ш§Щ„ Ш§ЫҢЩҶ Щ…Ъ©Ш§ЩҶЫҢШІЩ… ШЁШұ ШұЩҲЫҢ ШҜШ§ШҜЩҮ ШЁШ§ШІЪҜШұШҜШ§ЩҶШҜЩҮ ШҙШҜЩҮ ШӘЩ…Ш§Щ…ЫҢ APIЩҮШ§ Ш§ШІ Ш¬Щ…Щ„ЩҮ Ш®Ш·Ш§ЩҮШ§.
