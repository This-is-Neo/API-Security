    ЩҫЫҢШ§Щ…ШҜ Ш№ЩҲШ§Щ…Щ„ ШӘЩҮШҜЫҢШҜ / Щ…ШіЫҢШұ ШӯЩ…Щ„ЩҮ Ш¶Ш№ЩҒ Ш§Щ…ЩҶЫҢШӘЫҢ
    ШЁШ·ЩҲШұ Ъ©Щ„ЫҢ ШЁЩҮШұЩҮвҖҢШЁШұШҜШ§ШұЫҢ Ш§ШІ Ш§ЫҢЩҶ ШўШіЫҢШЁвҖҢЩҫШ°ЫҢШұЫҢ ЩҶШЁШ§ЫҢШҜ ШӘШЈШ«ЫҢШұШ§ШӘ ЩҒЩҶЫҢ ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙШҜ. Ш§Щ…Ш§ Щ…ЩҲШ§ШұШҜЫҢ Щ…Ш§ЩҶЩҶШҜ Ш№ШҜЩ… Ш§Щ…Ъ©Ш§ЩҶ Ш®ШұЫҢШҜ Щ…ШӯШөЩҲЩ„ ШӘЩҲШіШ· Ъ©Ш§ШұШЁШұШ§ЩҶ Щ…Ш№ШӘШЁШұ ЫҢШ§ Ш§ЫҢШ¬Ш§ШҜ ШӘЩҲШұЩ… ШҜШұ Ш§ЩӮШӘШөШ§ШҜ ШҜШ§Ш®Щ„ЫҢ ЩҶЫҢШІ Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ЩҫЫҢШ§Щ…ШҜ Ш§ЫҢЩҶ ШўШіЫҢШЁвҖҢЩҫШ°ЫҢШұЫҢ ШЁШ§ШҙЩҶШҜ. ЩҶШҜШ§ШҙШӘЩҶ ЫҢЪ© ШҜЫҢШҜЪҜШ§ЩҮ Ъ©Щ„ЫҢ Ш§ШІ API ШЁШұШ§ЫҢ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Ъ©Ш§Щ…Щ„ Ш§ШІ ЩҶЫҢШ§ШІЩҮШ§ЫҢ Ъ©ШіШЁ ЩҲ Ъ©Ш§Шұ ШЁЩҮ ШӘЪ©ШұШ§Шұ Ш§ЫҢЩҶ Щ…ШҙЪ©Щ„Ш§ШӘ Щ…ЩҶШ¬Шұ Щ…ЫҢвҖҢШҙЩҲШҜ. Щ…ЩҮШ§Ш¬Щ…Ш§ЩҶ ШЁЩҮ ШөЩҲШұШӘ ШҜШіШӘЫҢ Щ…ЩҶШ§ШЁШ№ ЩҮШҜЩҒ(Щ…Ш«Щ„Ш§ЩӢ ЩҶЩӮШ§Ш· ЩҫШ§ЫҢШ§ЩҶ) ЩҲ ЪҶЪҜЩҲЩҶЪҜЫҢ Ъ©Ш§ШұЪ©ШұШҜ ШўЩҶЩҮШ§ ШұШ§ Щ…ШҙШ®Шө Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜ. Ш§ЪҜШұ Щ…Ъ©Ш§ЩҶЫҢШІЩ…вҖҢЩҮШ§ЫҢ Щ…Ш®ШөЩҲШө Ш¬Щ„ЩҲЪҜЫҢШұЫҢ Ш§ШІ ШӯЩ…Щ„Ш§ШӘ (ШӘШ№ШҜШ§ШҜ ШҜШіШӘШұШіЫҢ Щ…ШӯШҜЩҲШҜ ШЁЩҮ APIШҢ Щ…ШӯШҜЩҲШҜЫҢШӘ ЩҶШұШ® ЩҲ ШәЫҢШұЩҮ) Ш§ШІ ЩӮШЁЩ„ ЩҲШ¬ЩҲШҜ ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙЩҶШҜШҢ Щ…ЩҮШ§Ш¬Щ…Ш§ЩҶ ШЁШ§ЫҢШҜ ШұШ§ЩҮЫҢ ШЁШұШ§ЫҢ ШҜЩҲШұ ШІШҜЩҶ ШўЩҶЩҮШ§ ЩҫЫҢШҜШ§ Ъ©ЩҶЩҶШҜ. ШЁЩҮШұЩҮ ШЁШұШҜШ§ШұЫҢ Ш§ШІ Ш§ЫҢЩҶ ШўШіЫҢШЁвҖҢЩҫШ°ЫҢШұЫҢ ШәШ§Щ„ШЁШ§ ЩҶЫҢШ§ШІ ШЁЩҮ ЩҒЩҮЩ… Щ…ЩҶШ·ЩӮ Ъ©ШіШЁ ЩҲ Ъ©Ш§ШұШҢ ШұЩҲШ§ШЁШ· Щ…Ш§ШЁЫҢЩҶ Ш§ШҙЫҢШ§ ЩҲ ШіШ§Ш®ШӘШ§Шұ API Ш§ШІ ШіЩҲЫҢ Щ…ЩҮШ§Ш¬Щ… ШҜШ§ШұШҜ.

    ШўЫҢШ§ API Ш§ШІ ЩҶШёШұ ШҜШіШӘШұШіЫҢ ШЁШҜЩҲЩҶ Щ…ШӯШҜЩҲШҜЫҢШӘ ШЁЩҮ Ш¬ШұЫҢШ§ЩҶвҖҢЩҮШ§ЫҢ Ъ©ШіШЁвҖҢЩҲЪ©Ш§Шұ ШӯШіШ§Ші вҖҢвҖҢвҖҢШўШіЫҢШЁвҖҢЩҫШ°ЫҢШұ Ш§ШіШӘШҹ

    ШҜШұ ШІЩ…Ш§ЩҶ Ш§ЫҢШ¬Ш§ШҜ ЫҢЪ© API EndpointШҢ ШЁШ§ЫҢШҜ Щ…ШҙШ®Шө ШҙЩҲШҜ ЪҶЩҮ Ш¬ШұЫҢШ§ЩҶ Ъ©Ш§ШұЫҢвҖҢШ§ЫҢ Ш§ЩҒШҙШ§ Щ…ЫҢвҖҢШҙЩҲШҜ. ШЁШұШ®ЫҢ Ш§ШІ Ш¬ШұЫҢШ§ЩҶвҖҢЩҮШ§ЫҢ Ъ©Ш§ШұЫҢ ЩҶШіШЁШӘ ШЁЩҮ ШҜЫҢЪҜШұШ§ЩҶ ШӯШіШ§ШівҖҢШӘШұ ЩҮШіШӘЩҶШҜШҢ ШЁЩҮ Щ…Ш№ЩҶШ§ЫҢ Ш§ЫҢЩҶЪ©ЩҮ ШҜШіШӘШұШіЫҢ ШЁЩҮ ШўЩҶЩҮШ§ ШЁЫҢШҙ Ш§ШІ ШӯШҜ Щ…Ш¬Ш§ШІ Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ШЁЩҮ Ъ©ШіШЁ ЩҲ Ъ©Ш§Шұ ШўШіЫҢШЁ ШЁШІЩҶШҜ.

    ЩҶЩ…ЩҲЩҶЩҮвҖҢвҖҢвҖҢЩҮШ§ЫҢЫҢ Ш§ШІ В«ЩҲЫҢЪҳЪҜЫҢвҖҢвҖҢвҖҢЩҮШ§ЫҢ ШӯШіШ§ШіВ» Ш№ШЁШ§ШұШӘЩҶШҜ Ш§ШІ:

    • Ш¬ШұЫҢШ§ЩҶ Ш®ШұЫҢШҜ Щ…ШӯШөЩҲЩ„ - Щ…ЩҮШ§Ш¬Щ… Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШЁЩҮ ЫҢЪ©вҖҢШЁШ§ШұЩҮ ШӘЩ…Ш§Щ… Щ…ЩҲШ¬ЩҲШҜЫҢ ЫҢЪ© Щ…ШӯШөЩҲЩ„ ШЁШ§ ШӘЩӮШ§Ш¶Ш§ЫҢ ШЁШ§Щ„Ш§ ШұШ§ Ш®ШұЫҢШҜШ§ШұЫҢ Ъ©ШұШҜЩҮ ЩҲ ШіЩҫШі ШўЩҶ Щ…ШӯШөЩҲЩ„ ШұШ§ ШЁШ§ ЩӮЫҢЩ…ШӘ ШЁШ§Щ„Ш§ШӘШұЫҢ Щ…Ш¬ШҜШҜШ§ЩӢ ШЁЩҒШұЩҲШҙШҜ (scalping).
    • Ш¬ШұЫҢШ§ЩҶ Ш§ЫҢШ¬Ш§ШҜ ЩҶШёШұ ЫҢШ§ ЩҫШіШӘ - Щ…ЩҮШ§Ш¬Щ… Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ШіЫҢШіШӘЩ… ШұШ§ ШЁШ§ Ш§ШұШіШ§Щ„ ЩҶШёШұШ§ШӘ ЫҢШ§ ЩҫШіШӘвҖҢЩҮШ§ЫҢ Щ…Ъ©ШұШұ ШҜЪҶШ§Шұ Щ…ШҙЪ©Щ„ Ъ©ЩҶШҜ.
    • Ш¬ШұЫҢШ§ЩҶ ШұШІШұЩҲ Ъ©ШұШҜЩҶ - Щ…ЩҮШ§Ш¬Щ… Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШӘЩ…Ш§Щ… ШЁШ§ШІЩҮвҖҢЩҮШ§ЫҢ ШІЩ…Ш§ЩҶЫҢ Щ…ЩҲШ¬ЩҲШҜ ШұШ§ ШұШІШұЩҲ Ъ©ШұШҜЩҮ ЩҲ Щ…Ш§ЩҶШ№ Ш§ШіШӘЩҒШ§ШҜЩҮ ШҜЫҢЪҜШұ Ъ©Ш§ШұШЁШұШ§ЩҶ ШҙЩҲШҜ.

    Ш®Ш·Шұ ШҜШіШӘШұШіЫҢ ШЁЫҢШҙ Ш§ШІ ШӯШҜШҢ ШЁЫҢЩҶ ШөЩҶШ§ЫҢШ№ ЩҲ Ъ©ШіШЁ ЩҲ Ъ©Ш§ШұЩҮШ§ЫҢ Щ…Ш®ШӘЩ„ЩҒ Щ…ШӘШәЫҢШұ Ш§ШіШӘ. ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ Щ…Ш«Ш§Щ„ШҢ Ш§ЫҢШ¬Ш§ШҜ ЩҫШіШӘвҖҢ ШӘЩҲШіШ· ЫҢЪ© Ш§ШіЪ©ШұЫҢЩҫШӘ Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ШҜШұ ЫҢЪ© ШҙШЁЪ©ЩҮ Ш§Ш¬ШӘЩ…Ш§Ш№ЫҢ ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ Ш®Ш·Шұ Ш§ШіЩҫЩ… ШҜШұ ЩҶШёШұ ЪҜШұЩҒШӘЩҮ ШҙЩҲШҜШҢ Ш§Щ…Ш§ ШҜШұШҙШЁЪ©ЩҮ Ш§Ш¬ШӘЩ…Ш§Ш№ЫҢ ШҜЫҢЪҜШұ ШӘШҙЩҲЫҢЩӮ ШҙЩҲШҜ.
    Ш§ЪҜШұ ЫҢЪ© ШӘШ§ШЁШ№ Ш§ЩҶШӘЩҮШ§ЫҢЫҢ API Ш§Щ…Ъ©Ш§ЩҶ ШҜШіШӘШұШіЫҢ ШЁЫҢШҙ Ш§ШІ ШӯШҜ ШЁЩҮ ЫҢЪ© Ш¬ШұЫҢШ§ЩҶ Ъ©ШіШЁ ЩҲ Ъ©Ш§Шұ ШӯШіШ§Ші ШұШ§ ЩҒШұШ§ЩҮЩ… вҖҢЪ©ЩҶШҜШҢ ШҜШұ Щ…Ш№ШұШ¶ ШӯЩ…Щ„Ш§ШӘ ЩҲ ШіЩҲШЎШ§ШіШӘЩҒШ§ШҜЩҮвҖҢ Щ…ЩҮШ§Ш¬Щ…Ш§ЩҶ Ш®ЩҲШ§ЩҮШҜ ШЁЩҲШҜ.

    Щ…Ш«Ш§Щ„вҖҢвҖҢвҖҢЩҮШ§ЫҢЫҢ Ш§ШІ ШіЩҶШ§ШұЫҢЩҲЩҮШ§ЫҢ ШӯЩ…Щ„ЩҮ

    ШіЩҶШ§ШұЫҢЩҲ #1

    ЫҢЪ© ШҙШұЪ©ШӘ ЩҒЩҶШ§ЩҲШұЫҢ Ш§Ш№Щ„Ш§Щ… Щ…ЫҢвҖҢЪ©ЩҶШҜ Ъ©ЩҮ ЩӮШөШҜ ШҜШ§ШұШҜ ЫҢЪ© Ъ©ЩҶШіЩҲЩ„ ШЁШ§ШІЫҢ Ш¬ШҜЫҢШҜ ШұШ§ ШҜШұ ШұЩҲШІ ШҙЪ©ШұЪҜШІШ§ШұЫҢ Щ…ЩҶШӘШҙШұ Ъ©ЩҶШҜ. Ш§ЫҢЩҶ Щ…ШӯШөЩҲЩ„ ШӘЩӮШ§Ш¶Ш§ЫҢ ШЁШіЫҢШ§Шұ ШЁШ§Щ„Ш§ ЩҲ Щ…ЩҲШ¬ЩҲШҜЫҢ Щ…ШӯШҜЩҲШҜЫҢ ШҜШ§ШұШҜ. Щ…ЩҮШ§Ш¬Щ… Ъ©ШҜЫҢ Щ…ЫҢвҖҢЩҶЩҲЫҢШіШҜ ШӘШ§ ШЁЩҮ ШөЩҲШұШӘ Ш®ЩҲШҜЪ©Ш§Шұ Щ…ШӯШөЩҲЩ„ Ш¬ШҜЫҢШҜ ШұШ§ ШЁШ®ШұШҜ.
    ШҜШұ ШұЩҲШІ Ш§ЩҶШӘШҙШ§ШұШҢ Щ…ЩҮШ§Ш¬Щ… Ъ©ШҜ ШұШ§ Ш§ШІ Ш·ШұЫҢЩӮ ШўШҜШұШі IPЩҮШ§ ЩҲ Щ…Ъ©Ш§ЩҶвҖҢЩҮШ§ЫҢ Щ…Ш®ШӘЩ„ЩҒ Ш§Ш¬ШұШ§ Щ…ЫҢвҖҢЪ©ЩҶШҜ. ШӘШ§ШЁШ№ Ш§ЩҶШӘЩҮШ§ЫҢЫҢ API Ш§ЩӮШҜШ§Щ…Ш§ШӘ ШӯЩҒШ§ШёШӘЫҢ Щ…ЩҶШ§ШіШЁЫҢ ШұШ§ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ ЩҶЪ©ШұШҜЩҮ ЩҲ ШҜШұЩҶШӘЫҢШ¬ЩҮ ШЁЩҮ Щ…ЩҮШ§Ш¬Щ… Ш§ЫҢЩҶ Ш§Щ…Ъ©Ш§ЩҶ ШұШ§ Щ…ЫҢвҖҢШҜЩҮШҜ Ъ©ЩҮ ШЁЫҢШҙШӘШұЫҢЩҶ ШӘШ№ШҜШ§ШҜ Щ…Щ…Ъ©ЩҶ Ш§ШІ Щ…ЩҲШ¬ЩҲШҜЫҢ ШұШ§ ЩӮШЁЩ„ Ш§ШІ ШіШ§ЫҢШұ Ъ©Ш§ШұШЁШұШ§ЩҶ Щ…Ш№ШӘШЁШұ ШЁШ®ШұШҜ.

    ШіЩҶШ§ШұЫҢЩҲ #2

    ЫҢЪ© ШҙШұЪ©ШӘ ЩҮЩҲШ§ЩҫЫҢЩ…Ш§ЫҢЫҢ Ш®ШҜЩ…Ш§ШӘ Щ…ШұШЁЩҲШ· ШЁЩҮ Ш®ШұЫҢШҜ ШЁЩ„ЫҢШ· ШўЩҶЩ„Ш§ЫҢЩҶ ШұШ§ ШЁШҜЩҲЩҶ ЩҮЫҢЪҶ ЪҜЩҲЩҶЩҮ ЩҮШІЫҢЩҶЩҮвҖҢЫҢ Щ„ШәЩҲ Ш®ШұЫҢШҜШҢ ШЁЩҮ Ъ©Ш§ШұШЁШұШ§ЩҶ Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ. ЫҢЪ© Ъ©Ш§ШұШЁШұШҢ 90ЩӘ Ш§ШІ ШөЩҶШҜЩ„ЫҢвҖҢЩҮШ§ЫҢ ЩҫШұЩҲШ§ШІ Щ…ЩҲШұШҜ ЩҶШёШұ ШұШ§ ШұШІШұЩҲ Щ…ЫҢвҖҢЪ©ЩҶШҜ.
    ЪҶЩҶШҜ ШұЩҲШІ ЩҫЫҢШҙ Ш§ШІ ЩҫШұЩҲШ§ШІШҢ Ъ©Ш§ШұШЁШұ Щ…Ш°Ъ©ЩҲШұШҢ ЩҮЩ…ЩҮ ШЁЩ„ЫҢШ·вҖҢЩҮШ§ ШұШ§ ЫҢЪ©вҖҢШ¬Ш§ Щ„ШәЩҲ Щ…ЫҢвҖҢЪ©ЩҶШҜШҢ Ъ©ЩҮ ШЁШ§Ш№Ш« Щ…ЫҢвҖҢШҙЩҲШҜ ШҙШұЪ©ШӘ ЩҮЩҲШ§ЩҫЫҢЩ…Ш§ЫҢЫҢ ШЁШұШ§ЫҢ ЩҫШұ Ъ©ШұШҜЩҶ ЩҫШұЩҲШ§ШІШҢ Щ…Ш¬ШЁЩҲШұ ШҙЩҲШҜ ШЁЩ„ЫҢШ·вҖҢЩҮШ§ ШұШ§ ШЁШ§ ШӘШ®ЩҒЫҢЩҒ ШЁЩҒШұЩҲШҙШҜ. ШҜШұ Ш§ЫҢЩҶ ШӯШ§Щ„ШӘШҢ Ъ©Ш§ШұШЁШұ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ЫҢЪ© ШЁЩ„ЫҢШ· ШЁЩҮ ЩӮЫҢЩ…ШӘ ШЁШіЫҢШ§Шұ Ш§ШұШІШ§ЩҶвҖҢШӘШұ Ш§ШІ ШЁЩ„ЫҢШ· Ш§ШөЩ„ЫҢ ШЁШ®ШұШҜ.

    ШіЩҶШ§ШұЫҢЩҲ #3

    ЫҢЪ© Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶ ШіЩҒШұ Ш§ШҙШӘШұШ§Ъ©ЫҢ ШЁШұЩҶШ§Щ…ЩҮвҖҢШ§ЫҢ ШЁШұШ§ЫҢ Щ…Ш№ШұЩҒЫҢ ШҜЩҲШіШӘШ§ЩҶ ШҜШ§ШұШҜ. Ъ©Ш§ШұШЁШұШ§ЩҶ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ ШҜЩҲШіШӘШ§ЩҶ Ш®ЩҲШҜ ШұШ§ ШҜШ№ЩҲШӘ Ъ©ШұШҜЩҮ ЩҲ ШЁШұШ§ЫҢ ЩҮШұ ШҜЩҲШіШӘЫҢ Ъ©ЩҮ ШЁЩҮ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶ ШЁЩҫЫҢЩҲЩҶШҜШҜШҢ Ш§Ш№ШӘШЁШ§Шұ ШҜШұЫҢШ§ЩҒШӘ Ъ©ЩҶЩҶШҜ. Ш§ЫҢЩҶ Ш§Ш№ШӘШЁШ§Шұ ШЁШ№ШҜШ§ЩӢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ ЩҲШ¬ЩҮ ЩҶЩӮШҜ ШЁШұШ§ЫҢ ШұШІШұЩҲ ШіЩҒШұЩҮШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ ШҙЩҲШҜ. Щ…ЩҮШ§Ш¬Щ… ШЁШ§ ЩҶЩҲШҙШӘЩҶ ЫҢЪ© Ш§ШіЪ©ШұЫҢЩҫШӘ ЩҒШұШўЫҢЩҶШҜ Ш«ШЁШӘвҖҢЩҶШ§Щ… ШұШ§ ШЁЩҮ ШөЩҲШұШӘ Ш®ЩҲШҜЪ©Ш§Шұ Ш§ЩҶШ¬Ш§Щ… Щ…ЫҢвҖҢШҜЩҮШҜ ЩҲ ШЁШ§ ЩҮШұ ЩҒШұШўЫҢЩҶШҜ Ш«ШЁШӘвҖҢЩҶШ§Щ… Ъ©Ш§ШұШЁШұ Ш¬ШҜЫҢШҜШҢ Ш§Ш№ШӘШЁШ§ШұЫҢ ШЁЩҮ Ъ©ЫҢЩҒ ЩҫЩҲЩ„Шҙ Ш§Ш¶Ш§ЩҒЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ. Щ…ЩҮШ§Ш¬Щ… ШЁШ№ШҜШ§ЩӢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ Ш§ШІ ШіЩҒШұЩҮШ§ЫҢ ШұШ§ЫҢЪҜШ§ЩҶ ШЁЩҮШұЩҮвҖҢШЁШұШҜШ§ШұЫҢ Ъ©ШұШҜЩҮ ЫҢШ§ ШӯШіШ§ШЁвҖҢЩҮШ§ЫҢЫҢ ШЁШ§ Ш§Ш№ШӘШЁШ§ШұЩҮШ§ЫҢ Ш§Ш¶Ш§ЩҒЫҢ ШұШ§ ШҜШұ Ш§ШІШ§ЫҢ ЩҫЩҲЩ„ ЩҶЩӮШҜ ШЁЩҒШұЩҲШҙШҜ.

    ЪҶЪҜЩҲЩҶЩҮ Ш§ШІ вҖҢвҖҢвҖҢШўШіЫҢШЁвҖҢЩҫШ°ЫҢШұЫҢ ШҜШіШӘШұШіЫҢ ШЁШҜЩҲЩҶ Щ…ШӯШҜЩҲШҜЫҢШӘ ШЁЩҮ Ш¬ШұЫҢШ§ЩҶвҖҢЩҮШ§ЫҢ Ъ©ШіШЁвҖҢЩҲЪ©Ш§Шұ ШӯШіШ§Ші ЩҫЫҢШҙЪҜЫҢШұЫҢ Ъ©ЩҶЫҢЩ…Шҹ

    ШЁШұЩҶШ§Щ…ЩҮвҖҢШұЫҢШІЫҢ ШЁШұШ§ЫҢ Ъ©Ш§ЩҮШҙ ШӘЩҮШҜЫҢШҜШ§ШӘ ШҜШұ ШҜЩҲ Щ„Ш§ЫҢЩҮ ШЁШ§ЫҢШҜ Ш§ЩҶШ¬Ш§Щ… ШҙЩҲШҜ:

    • ШҜШұ Щ„Ш§ЫҢЩҮ Ъ©ШіШЁ ЩҲ Ъ©Ш§ШұШҢ ШЁШ§ЫҢШҜ Ш¬ШұЫҢШ§ЩҶвҖҢЩҮШ§ЫҢ Ъ©ШіШЁ ЩҲ Ъ©Ш§Шұ ШӯШіШ§ШіЫҢ ШұШ§ ШҙЩҶШ§ШіШ§ЫҢЫҢ Ъ©ЩҶЫҢЩ… Ъ©ЩҮ Ш§ЪҜШұ ШЁЩҮ ШөЩҲШұШӘ ЩҶШұЩ…вҖҢШ§ЩҒШұШ§ШІЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ ШҙЩҲЩҶШҜШҢ Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ШЁЩҮ Ъ©ШіШЁвҖҢЩҲЪ©Ш§Шұ ШўШіЫҢШЁ ШЁШІЩҶЩҶШҜ.
    • ШҜШұ Щ„Ш§ЫҢЩҮ Щ…ЩҮЩҶШҜШіЫҢШҢ Щ…Ъ©Ш§ЩҶЫҢШІЩ…вҖҢЩҮШ§ЫҢ ШӯЩҒШ§ШёШӘЫҢ Щ…ЩҶШ§ШіШЁЫҢ ШұШ§ ШЁШұШ§ЫҢ Ъ©Ш§ЩҮШҙ Ш®Ш·ШұЩҮШ§ЫҢ Щ„Ш§ЫҢЩҮ Ъ©ШіШЁ ЩҲ Ъ©Ш§Шұ Ш§ЩҶШӘШ®Ш§ШЁ Щ…ЫҢвҖҢЪ©ЩҶЫҢЩ….

    ШҜШұ Ш§ЫҢЩҶ ЩӮШіЩ…ШӘ ШЁЩҮ Щ…Ъ©Ш§ЩҶЫҢШІЩ…вҖҢЩҮШ§ЫҢ ШӯЩҒШ§ШёШӘЫҢ Щ…Ш®ШӘЩ„ЩҒ ШЁШұШ§ЫҢ Ъ©Ш§ЩҮШҙ ШӘЩҮШҜЫҢШҜШ§ШӘ Ш®ЩҲШҜЪ©Ш§Шұ Ш§ШҙШ§ШұЩҮ ШҙШҜЩҮ Ш§ШіШӘ. ШЁШұШ®ЫҢ Ш§ШІ Ш§ЫҢЩҶ Щ…Ъ©Ш§ЩҶЫҢШІЩ…вҖҢЩҮШ§ ШіШ§ШҜЩҮвҖҢШӘШұ ЩҮШіШӘЩҶШҜ ЩҲ ШЁШұШ®ЫҢ ШҜЫҢЪҜШұ ЩҫЫҢЪҶЫҢШҜЩҮвҖҢШӘШұ. ШұЩҲШҙвҖҢЩҮШ§ЫҢ Щ…Ш®ШӘЩ„ЩҒЫҢ ШЁШұШ§ЫҢ Ъ©Ш§ЩҮШҙ ШіШұШ№ШӘ ШӘЩҮШҜЫҢШҜШ§ШӘ Ш®ЩҲШҜЪ©Ш§Шұ Щ…ЩҲШұШҜ Ш§ШіШӘЩҒШ§ШҜЩҮ ЩӮШұШ§Шұ Щ…ЫҢвҖҢЪҜЫҢШұШҜ:

    • ШҙЩҶШ§ШіШ§ЫҢЫҢ ШҜШіШӘЪҜШ§ЩҮ: Ш§ЫҢЩҶ ШұЩҲШҙ Ш§ШІ Ш·ШұЫҢЩӮ ШҙЩҶШ§ШіШ§ЫҢЫҢ ЩҲ Щ…Щ…ЩҶЩҲШ№ЫҢШӘ ШҜШіШӘШұШіЫҢ ШЁЩҮ ШҜШіШӘЪҜШ§ЩҮвҖҢЩҮШ§ЫҢ ЩҶШ§ШҙЩҶШ§Ш®ШӘЩҮ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ Щ…ЩҮШ§Ш¬Щ…Ш§ЩҶ ШұШ§ ЩҲШ§ШҜШ§Шұ ШЁЩҮ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШұШ§ЩҮЪ©Ш§ШұЩҮШ§ЫҢ ЩҫЫҢЪҶЫҢШҜЩҮвҖҢШӘШұЫҢ Ъ©ЩҶШҜ Ъ©ЩҮ ШЁШұШ§ЫҢ ШўЩҶЩҮШ§ ЩҮШІЫҢЩҶЩҮ ШЁЫҢШҙШӘШұЫҢ ШҜШ§ШұШҜ. Щ…Ш«Щ„Ш§ЩӢШҢ ШіЫҢШіШӘЩ… Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ШҜШіШӘШұШіЫҢ Щ…ШұЩҲШұЪҜШұЩҮШ§ЫҢ ШЁШҜЩҲЩҶ ШұШ§ШЁШ· Ъ©Ш§ШұШЁШұЫҢ ШұШ§ Щ…Щ…ЩҶЩҲШ№ Ъ©ЩҶШҜ.
    • ШҙЩҶШ§ШіШ§ЫҢЫҢ Ш§ЩҶШіШ§ЩҶ: Ш§ШІ ШұШ§ЩҮЪ©Ш§ШұЩҮШ§ЫҢЫҢ Щ…Ш§ЩҶЩҶШҜ Captcha ЫҢШ§ ШұШ§ЩҮЪ©Ш§ШұЩҮШ§ЫҢ ШЁЫҢЩҲЩ…ШӘШұЫҢЪ© ЩҫЫҢШҙШұЩҒШӘЩҮвҖҢШӘШұ Щ…Ш§ЩҶЩҶШҜ Ш§Щ„ЪҜЩҲЩҮШ§ЫҢ ШӘШ§ЫҢЩҫ Ъ©ШұШҜЩҶ ШЁШұШ§ЫҢ ШҙЩҶШ§ШіШ§ЫҢЫҢ Ъ©Ш§ШұШЁШұШ§ЩҶ Ш§ЩҶШіШ§ЩҶЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ.
    • Ш§Щ„ЪҜЩҲЩҮШ§ЫҢ ШәЫҢШұШ§ЩҶШіШ§ЩҶЫҢ: ШЁШ§ ШӘШ¬ШІЫҢЩҮ ЩҲ ШӘШӯЩ„ЫҢЩ„ Ш§Щ„ЪҜЩҲЩҮШ§ЫҢ Ш№Щ…Щ„Ъ©ШұШҜ Ъ©Ш§ШұШЁШұШ§ЩҶ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ Ш§Щ„ЪҜЩҲЩҮШ§ЫҢ ШәЫҢШұШ§ЩҶШіШ§ЩҶЫҢ ШұШ§ ШҙЩҶШ§ШіШ§ЫҢЫҢ Ъ©ШұШҜ. ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ Щ…Ш«Ш§Щ„ШҢ ШҜШіШӘШұШіЫҢ Ъ©Ш§ШұШЁШұ ШЁЩҮ Ш№Щ…Щ„ЫҢШ§ШӘ "Ш§ЩҒШІЩҲШҜЩҶ ШЁЩҮ ШіШЁШҜ Ш®ШұЫҢШҜ" ЩҲ "ШӘЪ©Щ…ЫҢЩ„ Ш®ШұЫҢШҜ" ШҜШұ Ъ©Щ…ШӘШұ Ш§ШІ ЫҢЪ© Ш«Ш§ЩҶЫҢЩҮШҢ Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ЩҶШҙШ§ЩҶЩҮвҖҢШ§ЫҢ Ш§ШІ Ш§Щ„ЪҜЩҲЫҢ ШәЫҢШұШ§ЩҶШіШ§ЩҶЫҢ ШЁШ§ШҙШҜ.
    • Щ…ШіШҜЩҲШҜ Ъ©ШұШҜЩҶ ШўШҜШұШівҖҢЩҮШ§ЫҢ IP Ш§ШІ ЪҜШұЩҮвҖҢЩҮШ§ЫҢ Ш®ШұЩҲШ¬ЫҢ Tor ЩҲ ЩҫШұЩҲЪ©ШіЫҢвҖҢЩҮШ§ЫҢ Щ…Ш№ШұЩҲЩҒ: Ш§ЫҢЩҶ ШұЩҲШҙ ШЁЩҮ Щ…ШіШҜЩҲШҜ Ъ©ШұШҜЩҶ ШўШҜШұШівҖҢЩҮШ§ЫҢ IP Щ…Ш®ШөЩҲШө Щ…ЫҢвҖҢЩҫШұШҜШ§ШІШҜ Ъ©ЩҮ Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ШӘЩҲШіШ· Щ…ЩҮШ§Ш¬Щ…Ш§ЩҶ Щ…ЩҲШұШҜ Ш§ШіШӘЩҒШ§ШҜЩҮ ЩӮШұШ§Шұ ЪҜЫҢШұШҜ.
    • Щ…ШӯШҜЩҲШҜ Ъ©ШұШҜЩҶ ШҜШіШӘШұШіЫҢ ШЁЩҮ APIвҖҢЩҮШ§ЫҢ Щ…ШөШұЩҒЫҢ Щ…ШіШӘЩӮЫҢЩ… ШӘЩҲШіШ· ШҜШіШӘЪҜШ§ЩҮвҖҢЩҮШ§ (Щ…Ш§ЩҶЩҶШҜ APIвҖҢЩҮШ§ЫҢ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ ЩҲ B2B) Щ…ЩҮШ§Ш¬Щ…Ш§ЩҶ ШұШ§ Ш§ШІ ШҜШіШӘШұШіЫҢ ШўШіШ§ЩҶ ШЁЩҮ Ш§ЫҢЩҶ APIвҖҢЩҮШ§ ШЁШ§ШІЩ…ЫҢвҖҢШҜШ§ШұШҜ. Ш§ШІ ШўЩҶвҖҢШ¬Ш§ЫҢЫҢ Ъ©ЩҮ Ш§ЫҢЩҶ ЩҶЩҲШ№ APIвҖҢЩҮШ§ Ш§ШәЩ„ШЁ ШӘЩ…Ш§Щ… Щ…Ъ©Ш§ЩҶЫҢШІЩ…вҖҢЩҮШ§ЫҢ ШӯЩҒШ§ШёШӘЫҢ Щ…ЩҲШұШҜ ЩҶЫҢШ§ШІ ШұШ§ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ ЩҶЩ…ЫҢвҖҢЪ©ЩҶЩҶШҜШҢ Щ…Ш№Щ…ЩҲЩ„Ш§ ШЁШұШ§ЫҢ Щ…ЩҮШ§Ш¬Щ…Ш§ЩҶ ЩҮШҜЩҒ ШўШіШ§ЩҶЫҢ Щ…ЫҢвҖҢШЁШ§ШҙЩҶШҜ.
